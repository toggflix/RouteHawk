# Screenshot Policy

Real screenshot files can be placed in this directory.

Do not use fake, placeholder, edited, or third-party target screenshots.

Recommended screenshot names:

- `dashboard-latest-run.png`
- `dashboard-compare-drilldown.png`
- `html-report-findings.png`
- `endpoint-inventory-filters.png`

Before taking screenshots, use the local demo lab.

Screenshots must not show real target data, private data, credentials, tokens, third-party sites, or unauthorized scan results.
